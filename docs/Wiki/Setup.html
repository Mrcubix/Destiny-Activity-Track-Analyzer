Creating A New Bungie Application

  • Log into your Bungie account.

  • Navigate to the Your Applications Page.

  • Click on Create New App.

  • Give the application a fairly descriptive name.

    Recommended: Destiny Activity Track Analyzer

  • Do not touch anything else and just go down past the last section.

    You should see a checkbox and a Create New App Button.

    Read the Term of Use if you're into that stuff.

  • Click on Create New App.

  • Head to the API KEYS Section.

    There you will find your API Key, you will need to copy it

    (Don't try using this key, i removed it)

  • Enter this API Key in DATA and click on Proceed.

  • If you did everything properly, there shouldn't be any errors.
